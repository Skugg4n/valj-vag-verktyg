# Product Vision

## One-Liner
Ett kreativt verktyg för att skapa Choose Your Own Adventure-böcker med sömlös växling mellan nodvy, linjär textvy och spelvy.

## Problem
Att skriva interaktiva berättelser (CYOA) kräver idag antingen komplexa programmeringsverktyg eller klumpiga textdokument utan överblick. Författare saknar ett intuitivt sätt att samtidigt se strukturen (vilka vägar som finns) och skriva texten (i långform).

## Vision
En sömlös kreativ upplevelse där författare kan:
- **Se helheten** i en nodvy med kopplingar och vägval
- **Skriva flödande text** i en linjär vy som känns som Google Docs
- **Läsa resultatet** i en play-vy som simulerar läsarens upplevelse
- **Exportera** som markdown eller JSON med full backup av positioner, färger och metadata

Formatering i den linjära vyn synkar automatiskt med nodvyn — minimalt med manuellt arbete, maximalt med kreativ frihet.

## Core Principles
- **Sömlöshet** — växla mellan vyerna utan att tappa kontext eller data
- **Lätthet** — extremt enkla medel för att formatera och strukturera
- **Överblick** — alltid kunna se hur berättelsens delar hänger ihop
- **Portabilitet** — export i öppna format (Markdown, JSON)

## Non-Goals
- Inte en publiceringsplattform — vi skapar, vi publicerar inte
- Inte en generell texteditor — fokus är CYOA-berättelser
- Inte multiplayer/co-editing (i nuläget)
