Help

This tool lets you build interactive choose-your-path stories. Each node represents a section of the story and can link to other nodes using Markdown references.

Creating Nodes

  • Click on an empty area of the graph to create a new node.
  • Edit the title and body text directly inside the node.
  • Type #001 to create a reference or use Markdown links such as [Continue â†’ #002](#002).

Managing Projects

  • Enable Save next to the project name to store the current story locally.
  • Use the floating menu in the bottom right to export or import projects.
  • Select Export MD in the header to download all nodes as a Markdown file.

Playthrough

  • Choose Play from the floating menu to read through the story starting from the current node.

AI Features

  • Configure API settings under AI Settings in the floating menu.
  • The cloud icon generates suggestions for how the story might continue.
  • The spell check icon proofreads the current text.
  • The color square lets you change the background of the selected node.
