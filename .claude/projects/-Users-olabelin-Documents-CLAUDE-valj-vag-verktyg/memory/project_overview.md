Välj Väg Verktyg is a React + Vite branching story editor for Swedish children's books (CYOA-style). Deployed at valj-vag-verktyg.vercel.app, repo at Skugg4n/valj-vag-verktyg.

**Why:** Tool for writing interactive/branching narratives with a visual node graph and linear reading view.

**How to apply:** The app is used for authoring Swedish-language stories. UI text can be Swedish or English. Key features: React Flow graph editor, Linear View, Playthrough mode, TipTap rich text, project save/load via localStorage, import/export JSON.
